K8s 기반 컨테이너 구성 및 서버 모니터링 프로젝트

개요:
이 프로젝트는 쿠버네티스(K8s) 기반의 컨테이너 환경을 구축하고, 효과적인 서버 모니터링 시스템을 구축하는 것을 목표로 합니다. 

주요 구성요소:
1. 컨테이너 오케스트레이션: Kubernetes 클러스터 설정
2. 모니터링 시스템: Prometheus + Grafana 스택
3. 로그 수집: ELK Stack (Elasticsearch, Logstash, Kibana)
4. 알람 시스템: AlertManager를 통한 실시간 알림

기대 효과:
- 서비스 가용성 향상
- 리소스 사용률 최적화  
- 실시간 장애 감지 및 대응
